How to Use the Build Planner

  1. Choose one of Mina's five main weapons.
  2. Select the Sidearm used for the build.
  3. Choose up to six Trinkets.
  4. Review the generated build summary.
  5. Copy the URL to save or share the setup.

What Makes a Good Build?

A useful build should have a clear purpose. Boss builds may prioritize damage, Joule efficiency, and survival, while exploration builds often benefit from movement, resource recovery, and safer traversal.

Boss builds

Boss-focused setups usually combine a comfortable main weapon, a Sidearm that covers range or burst damage, and Trinkets that improve damage, healing, defense, or resource efficiency.

Exploration builds

Exploration setups can prioritize movement, pit safety, repeated burrowing, recovery, and convenient travel through older regions.

100% cleanup builds

Cleanup builds should focus on consistency rather than maximum damage. Choose equipment that makes revisiting rooms, fighting normal enemies, and checking hidden routes less frustrating.

Mina the Hollower Build Planner FAQ

Does the build planner save my setup?

The selected build is encoded in the generated URL. Copy or bookmark the URL to save the setup, and opening that URL will restore the selected weapon, Sidearm, and Trinkets.

How many Trinkets can I select?

The planner allows up to six Trinkets. When six are selected, the remaining unselected options are temporarily disabled.

What is the best Mina the Hollower build?

There is no single best build for every situation. Boss fights, exploration, achievement cleanup, and challenge runs benefit from different combinations.

Does the planner calculate damage?

No. This version creates and shares equipment combinations. It does not calculate exact damage, defense, or Joule efficiency.

Can I share a build on Reddit?

Yes. Generate a build, then use the Reddit button or copy the URL and paste it into a discussion.
